# WMacLib 
![image](https://raw.githubusercontent.com/Wicikk/WMacLib/main/Misc/Demo.png)
---
**WMacLib** is a fork of the original [MacLib](https://github.com/biggaboy212/Maclib) UI library with bug fixes and misc.

**[Documentation](https://github.com/Wicikk/WMacLib/blob/main/Documentation.md)** 


## Demo

```lua
local WMacLib = loadstring(game:HttpGet("https://raw.githubusercontent.com/Wicikk/WMacLib/main/WMacLib.lua"))()
WMacLib:Demo()
```

Loads a full demo window showcasing all elements.
